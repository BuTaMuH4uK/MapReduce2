Для начала, я реализовал скрипт MapScript, который выделяет слова из текста и приписывает им значение равное 1.Я реализовал только для латинских символов, но аналогично реализуется и для киррилицы, поэтому я не стал этим заниматься. Мой MapScript принимает java программа MapReduce, которая посылает map'у файл и принимает от него ответ в виде файла, который я далее обрабатываю. В виде обработки я имею ввиду внесение всех слов в list, который далее я сортирую и уже сортированный list, я вывожу в файл, который следом передаю скрипту ReduceScript. Этот скрипт складывает значения всех слов, и выводит в окончательный файл ans слово и число его встреч в тексте.
